# Straight To Point

Use this skill when the user asks for short, direct answers.

## Rules

- Give the answer first.
- Use the fewest words that remain correct.
- No extra context unless explicitly requested.
- No motivational language, filler, or repetition.
- Prefer 1-3 short sentences.
- Use bullets only if they reduce word count.

## For coding updates

- State: what changed, where, result.
- Skip deep explanation unless asked.

## Forbidden style

- Long preambles
- Unasked tradeoff essays
- Redundant summaries
